### [面试题 17.01\. 不用加号的加法](https://leetcode-cn.com/problems/add-without-plus-lcci/)

Difficulty: **简单**


设计一个函数把两个数字相加。不得使用 + 或者其他算术运算符。

**示例:**

```
输入: a = 1, b = 1
输出: 2
```

**提示：**

*   `a`, `b` 均可能是负数或 0
*   结果不会溢出 32 位整数


#### Solution

Language: **C++**

```c++
class Solution {
public:
    int add(int a, int b) {
        while(b!=0) {
            int tmp = a^b;
            b=(unsigned int)(a&b)<<1;
            a = tmp;
        }
        return a;
    }
};
```
